# Create React App with PNPM and Vite

# 01. Install node.js

 - Follow below link and download the node.js application. *(LTS Recommended)*
 [Link](https://nodejs.org/en)
  - Install node.js to the PC
  - Check its with ``` node -v ``` command. (Its display node version)

# 02. Install *PNPM* package manager with power shell

- *(We can use npm, yarn and bun package managers. But PNPM is recommended for this project)*

 - Copy below code and past in to the Windows PowerShell and hit enter
```
iwr https://get.pnpm.io/install.ps1 -useb | iex
```

#### For other OS : [Click here.](https://pnpm.io/installation)

# 03. Create React App with Vite Build Tool

 - Create folder for the project and open terminal.
 - Type ``` pnpm create vite ``` and press enter.
 - Then enter any name for the project *(see the below screenshot)*

 ![Screenshot 1](https://github.com/ShanelkaPramuditha/tutorials/blob/main/assets/images/Create-React-App-with-PNPM-and-Vite/Pasted%20image%2020231228004820.png?raw=true)
 
- Then select framework as "React"

![Screenshot 2](https://github.com/ShanelkaPramuditha/tutorials/blob/main/assets/images/Create-React-App-with-PNPM-and-Vite/Pasted%20image%2020231228005117.png?raw=true)

- Then select "javascript" and hit Enter.
- You can run project as below
